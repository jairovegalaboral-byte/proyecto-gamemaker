Nombre del juego: rpg tutorial

justificacion de la eleccion del nombre: es un juego basado en un tutorial que se seguira trabajando en el

Objetivo del juego:
matar a todos los enemigos y lograr subir de nivel paso a paso

Instrucciones:
el personaje principal se mueve con las teclas WSDA y ataca con el teclado

-epacio requerido por el momento 30mb(SE VERA EN LAS PROXIMAS ACTUALIZACIONES)
-recursos especificos: Visual C++ Redistributable
